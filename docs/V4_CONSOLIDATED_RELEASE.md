# Full Consolidated Release v4.8

v4.8 consolidates the repo around source-of-truth, schema validation, component/backend registries, cleanup planning, and release manifesting.

## What this release does

- Keeps frontend source-of-truth fixed.
- Adds UI component boundary registry.
- Adds backend module boundary registry.
- Adds schema-covered architecture outputs.
- Adds dead-file cleanup plan.
- Adds consolidated release manifest.

## What this release does not do

- It does not enable automatic trading.
- It does not convert expected-return priors into official alpha.
- It does not enable max-Sharpe rebalancing.
- It does not delete files automatically.
- It does not rewrite runtime UI/backend behavior.
