跳躍擴散指標解釋 (Jump-Diffusion Indicator Guide)
跳躍擴散 VaR 95 (JD VaR 95)： 在跳躍擴散模型下，95% 信心水準的一期虧損門檻；數值越高，代表短期左尾風險越重。
跳躍擴散 ES 95 (JD ES 95)： 跌破 VaR 95 門檻後，最壞 5% 情境的平均虧損；通常比 VaR 更能反映真正尾部損失。
崩盤機率 (Crash Prob)： 模型估計發生跳躍式大跌的機率；不是預測明天會崩盤，而是衡量斷崖式下跌風險強弱。
尾部損失 (Tail Loss)： 跳躍事件發生時的估計損失幅度；可視為「一旦出事，可能掉多少」的壓力 proxy。
年化跳躍頻率 λ (Annual Freq)： 一年中預期跳躍事件次數；λ=0.5 約等於平均兩年一次，λ=2 約等於一年兩次。
估計期間 (Horizon)： 這組風險估計對應的觀察或預測期間；期間越長，尾部風險通常越容易放大。
跳躍特徵 (Jump Signature)： μj 是平均跳幅，σj 是跳幅波動；μj 越負代表跳躍偏向下跌，σj 越大代表跳躍事件更不穩定。
EVT / POT-GPD 暫停顯示：目前使用週資料，尾部超額樣本不足；等改成「持倉標的長期日報酬 + 目前權重合成 portfolio return」後再啟用。

記帳 SOP

不要把內部調倉記成外部現金流

正確記法
  • 1. 銀行匯進投資帳戶 → Deposit
  • 2. 用帳上資金買股票 / ETF / 幣 → Buy
  • 3. 賣掉部位、錢先留在投資帳戶 → Sell
  • 4. 從投資帳戶領回銀行 / 生活帳戶 → Withdraw
硬規則
  • • Buy ≠ Deposit
  • • Sell ≠ Withdraw
  • • 換股 / 調倉不算外部現金流
  • • 先賣再出金；先入金再買入
  • • 每週固定按一次快照，不要亂補差額
常見錯法
  • • 把買入記成入金 → 成本與倉位會失真
  • • 把賣出記成出金 → 持股不會正確減少
  • • 用 History 手動補外部現金流 → TWR / MWR 會歪
